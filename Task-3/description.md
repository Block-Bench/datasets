# Task 3: Cross-Chain Token Bridge

## Domain Knowledge Focus
This task tests domain knowledge on **cross-chain protocols**, **LayerZero integration**, and **access control mechanisms**.

## Main Aspects
- **Cross-Chain Token Bridging**: Implements an OFT (Omnichain Fungible Token) contract that enables token transfers across multiple blockchain networks
- **LayerZero Protocol Integration**: Uses LayerZero's OFT standard for cross-chain messaging and token bridging
- **Transfer Restrictions**: Implements complex transfer validation logic that restricts bridged token holders and applies chain-specific rules (e.g., Arbitrum restrictions)
- **Access Control**: Manages owner privileges, transfer-allowed contracts, and bridged token holder tracking
- **Token Minting/Burning**: Handles token minting on destination chains and tracks bridged token recipients

## Key Technical Concepts
- Cross-chain protocols (LayerZero)
- Omnichain token standards (OFT)
- Transfer restriction logic
- Access control patterns
- Chain-specific validation
- Token bridging mechanisms
